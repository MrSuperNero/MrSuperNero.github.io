Forms

Examples of forms that I've seen

This is a form from Facebook's website. Some information they ask for include:

  • First Name
  • Last Name
  • Birthday
  • Desired Password
  • Gender

This form seems a bit dense with the small space between the input fields.

This is a form from Hulu's website. Some information they ask for include:

  • Email
  • Password
  • Name
  • Birthdate
  • Gender

I really like this registration field because it's spaced out and feels really clean. There is no clutter and everything is simple.

This is a form from Gmail's website. Some information they ask for include:

  • First Name
  • Last Name
  • Username for email
  • Password

This one is also rather simple. While it does not feel as cluttered as Facebook's registration section, it doesn't feel as clean as Hulu's.

This is a form from Zwift's website. Some information they ask for include:

  • First Name
  • Last Name
  • Email
  • Desired Password

The input fields are rather large and the placeholder text seem to blend in with the background, which is somewhat hard to read. The placeholders are also not vertically centered, which I believe they should be. They also do not ask for much information, such as birthday, but they simply have a button to ask if the user is older than 16 years old. The caption text below "Create An Account" also seems unnecessary and just clutters up the form.
